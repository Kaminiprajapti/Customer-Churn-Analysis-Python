# Customer Churn Analysis | Python

## Project Overview
Telco company ke 7,043 customers ka data 
analyze kiya — kyun customers company chhod 
rahe hain aur kaun zyada risk mein hai.

## Tools Used
- Python
- Pandas (Data Cleaning & Analysis)
- Matplotlib & Seaborn (Visualization)

## Dataset
- Source: Kaggle — Telco Customer Churn
- Rows: 7,043 | Columns: 21

## Key Insights
1. Overall churn rate **26.58%** hai — 
   har 4 mein se 1 customer ja raha hai
2. Month-to-month contract wale **42.71%** 
   churn karte hain vs 2-year wale sirf 2.85%
3. Churned customers avg **₹74.44/month** 
   dete the vs non-churned **₹61.31/month**
4. Naye customers (0-12 months) mein 
   **47.68%** churn rate — sabse high!

## Recommendations
1. Month-to-month customers ko long-term 
   contracts offer karo — discounts do
2. High charges wale customers ko loyalty 
   benefits do
3. Naye customers ke liye special onboarding 
   program banao — pehle 6 months critical hai
